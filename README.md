# AI-Resume-Analyzer-Tool

An AI-powered Resume Analysis platform that helps job seekers improve their resumes through ATS-based evaluation and personalized recommendations.

## Live Demo
https://truthful-smart-resume-path.base44.app

## Features

- Resume Upload and Analysis
- ATS Score Generation
- Skill Gap Identification
- Resume Improvement Suggestions
- User-Friendly Interface
- AI-Powered Recommendations

## Problem Statement

Many candidates get rejected because their resumes are not optimized for Applicant Tracking Systems (ATS). This project helps users identify weaknesses in their resumes and improve their chances of getting shortlisted.

## Solution

The AI Resume Analyzer evaluates resumes based on multiple factors such as:

- Resume Structure
- Skills Coverage
- ATS Compatibility
- Content Quality
- Improvement Suggestions

## Technologies Used

- Base44
- Artificial Intelligence
- Prompt Engineering
- Web Application Development

## Workflow

1. Upload Resume
2. AI Processes Resume
3. ATS Score is Generated
4. Detailed Feedback is Provided
5. User Improves Resume
